# ⚡ BlackLotus Ver

**رابط کاربری حرفه‌ای و مدرن** برای ابزار Domain Fronting گوگل

![BlackLotus GUI](https://github.com/Expen1/BlackLotus/blob/main/screenshot.png)

---

### معرفی
**BlackLotus** نسخهٔ بهبود یافته و کاملاً گرافیکی از پروژهٔ **[MasterHttpRelayVPN](https://github.com/masterking32/MasterHttpRelayVPN)** است.

با الهام از پروژهٔ اصلی masterking32 و تمرکز بر تجربه کاربری، این نسخه شامل رابط کاربری شیک، نمایش گرافیکی مسیر ترافیک، تولید رمز قوی، تست اتصال و امکانات حرفه‌ای دیگر شده است.

---

### ✨ ویژگی‌های کلیدی

- **رابط کاربری مدرن Dark Theme** با انیمیشن و طراحی حرفه‌ای
- **نمایش گرافیکی زنده مسیر ترافیک** (Browser → Proxy → Google → Relay → Target)
- **تولید رمز Auth Key قوی** به صورت اتوماتیک
- **دکمه تست اتصال** برای جلوگیری از ارور `unauthorized`
- **GIF آموزشی** داخل برنامه
- **اسکن بهترین IP گوگل** و نصب گواهی CA
- **پشتیبانی کامل SOCKS5 + HTTP**
- **Load Balancing** با چندین Script ID
